Roadmap

Suggestions for other features are welcome. Offers of help with use cases, examples and testing would be very much appreciated. Issues raised by users generally take precedence.

Features planned for the 1.1 release

  • Support global parameter declarations at the top level of the resources script.
  • Support forReading and forWriting attributes for sx:recordDelimiter.
  • Add sx:fallback element.
  • Enhancements to error handling.
  • Fix for reported bug "line number tracking incorrect for comment lines"

Features planned for subsequent releases

  • An XML schema for the ServingXML vocabulary
  • Support xml type in parameters and fields.
  • Enhancement to flat file signature elements to avoid buffering file in memory for signatures written to trailer records.
  • Add an installer
  • Extensions
    • An Apache POI extension to support reading and writing Excel worksheets
    • Incorporation of web service components contributed by Robin Keith.
  • Eclipse plug-in support.
  • Maven support.
